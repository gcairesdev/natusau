## Pré requisitos
* [git](https://git-scm.com/)
* [gerenciador de pacotes](#)
    * [npm](https://nodejs.org/en/)
    * [yarn](https://yarnpkg.com/)

## Instalação

Clone o repositório:

```git
git clone https://github.com/gcairesdev/natusau.com
```

Instale as dependências

```sh
npm i
```

Rode o projeto:

```sh
npm run dev
```
